Online Store

As the name implies, this is an online store (for video games). It is both the front-end and the back-end of an online store.

The front-end was written with HTML and CSS and makes use of Bootstrap 4 and AOS. The back-end was written with PHP and SQL. The website safely stores user and product information in a database and keeps track of orders. All this information is viewable in a menu only accessible to an ADMIN account. Here products can be added, removed or edited.

.../Games/JINC JUMP.exe

JINC JUMP

This is a simple highscore game about jumping up to procedurally generated platforms while the screen automatically moves up. For every 200m the player goes up, a new obstacle is introduced to give the game a natural difficulty curve.

The whole game was written with Python using the PyGame module (this includes all graphics).
